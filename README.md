# Same-chat-api

## Requirements

- Nodejs
- Docker (optional)

## Installing Dependencies

```
npm i
```

## Running Development

<br>

### Outside of the container

<br/>

```
npm run dev
```

<br/>

### Inside the container

<br/>
<br/>
Development:
<br/>
<br/>

```
npm run dc
```

<br/>

Production:

```
npm run pc
```
